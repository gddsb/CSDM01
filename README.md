# 奶粉罐生产管理系统 (Milk Can MES)

基于 React + Ant Design + Vite 构建的奶粉罐生产制造执行系统，涵盖生产管理、质量管理、设备管理、数据大屏及报表中心等核心业务模块。

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| React | 18.3 | 前端框架 |
| Ant Design | 5.21 | UI 组件库 |
| React Router | 6.26 | 路由管理 |
| Vite | 5.4 | 构建工具 |
| Day.js | 1.11 | 日期处理 |

## 功能模块

### 系统管理
- 用户管理：用户增删改查、角色分配、启用/禁用
- 角色权限：角色定义与权限管理
- 数据库字典：数据表结构与字段定义查看
- 系统配置：系统参数配置
- 操作日志：用户操作行为审计

### 基础数据
- 料品档案：料品基础信息及安全库存管理
- 产线管理：生产线配置与状态管理
- 工序管理：生产工序定义与顺序调整
- 制程检验不良分类：不良类型与单位定义

### 生产管理
- 生产订单：订单创建、下发、关闭，完工后查看关联工单/人员/异常记录
- 工单列表：工单全生命周期管理（开立→开工→关闭→完工），自动计算工时
- 生产报工：工序级报工，投入/产出/不良登记
- 人员记录：工单人员配置与班次管理，工单开工自动创建
- 异常记录：生产异常登记，关联订单与工单

### 质量管理
- 检验标准管理：检验标准定义与版本控制
- 来料检验 / 过程检验 / 成品检验：全流程质量检验
- 产品微生物检验 / 环境检验
- 投诉管理：客诉管理与供应商投诉
- 检测仪器管理：仪器校准记录

### 设备管理
- 设备档案：设备基础信息与特种设备检定
- 点检记录 / 维修保养 / 设备OEE

### 数据大屏
- 生产实时看板：工单进度与产线状态
- 质量分析看板
- 管理驾驶舱：综合数据展示

### 报表中心
- 生产日报 / 质量月报 / 效率分析
- 生产报表 / 质量报表 / 异常分析报表

## 工单状态流转

```
开立 → 开工 → 关闭 → 完工
```

| 状态 | 编辑 | 开工 | 删除 | 关闭 | 完工 |
|------|------|------|------|------|------|
| 开立 | ✓ | ✓ | ✓ | ✗ | ✗ |
| 开工 | ✗ | ✗ | ✗ | ✓ | ✗ |
| 关闭 | ✗ | ✗ | ✗ | ✗ | ✓ |
| 完工 | ✗ | ✗ | ✗ | ✗ | ✗ |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

开发服务器默认运行在 `http://localhost:5173`

## 项目结构

```
milk-can-mes/
├── src/
│   ├── assets/              # 静态资源（Logo图片）
│   ├── components/          # 公共组件（ThreeSectionPage等）
│   ├── contexts/            # React Context（用户、主题）
│   ├── layouts/             # 布局组件（MainLayout）
│   ├── mock/                # Mock 数据
│   ├── pages/               # 页面组件
│   │   ├── basic/           # 基础数据
│   │   ├── bigscreen/       # 数据大屏
│   │   ├── device/          # 设备管理
│   │   ├── production/      # 生产管理
│   │   ├── quality/         # 质量管理
│   │   ├── report/          # 报表中心
│   │   └── system/          # 系统管理
│   ├── styles/              # 全局样式
│   ├── themes/              # 主题配置
│   ├── AppContext.jsx       # 全局状态管理
│   ├── Dashboard.jsx        # 工作台首页
│   ├── Login.jsx            # 登录页
│   └── main.jsx            # 应用入口与路由
├── index.html
├── vite.config.js
└── package.json
```

## 功能特性

- 多主题切换（纯牛奶色、深色工厂等）
- 响应式布局，支持侧边栏折叠
- 表格表头居中、操作列统一宽度
- 右侧抽屉（Drawer）查看详情
- 工时按 0.5 小时取值计算
- 工单完工后关联记录自动转为只读
- 生产订单详情展示关联工单、人员、异常记录
